## O que é?

**Veiga Agenda** é um aplicativo web progressivo (PWA) gratuito para gerenciar compromissos, tarefas e lembretes com sincronização em tempo real entre todos os seus dispositivos via Google Firebase.

Funciona como ícone na tela inicial do iPhone, Android e como app no Windows — sem instalar nada, sem pagar nada.

## Funcionalidades

- Compromissos com data, hora, categoria e lembrete configurável
- Lista de tarefas com prioridade (alta, media, baixa) e prazo
- Visao consolidada do dia na aba "Hoje"
- Login com conta Google (OAuth2 via Firebase Auth)
- Sincronizacao em tempo real entre iPhone, Android e notebook
- Notificacoes de lembrete no navegador
- Visual NOC dark — interface limpa e responsiva
- 100% gratuito, dados privados por usuario

## Como instalar como app (PWA)

**iPhone (Safari)**
1. Acesse o link pelo Safari
2. Toque no botao de compartilhar
3. Selecione "Adicionar a Tela de Inicio"
4. Confirme — aparece como icone igual a um app nativo

**Android (Chrome)**
1. Acesse o link pelo Chrome
2. Toque nos tres pontos no canto superior direito
3. Selecione "Adicionar a tela inicial"

**Windows (Chrome ou Edge)**
1. Acesse o link
2. Clique no icone de instalar na barra de endereco
3. Confirme a instalacao

## Tecnologias

- HTML5 + CSS3 + JavaScript vanilla
- Firebase Authentication (login com Google)
- Firebase Firestore (banco de dados em tempo real)
- GitHub Pages (hospedagem gratuita)
- PWA (Progressive Web App) com suporte a instalacao

---

## Privacidade

Cada usuario ve apenas seus proprios dados. A autenticacao e feita via conta Google — nenhuma senha e armazenada no app. Os dados ficam no Firestore vinculados ao UID unico de cada usuario.
